# automation-scripts

### ssh connection automation
1. sshkeygen.sh - To generate ssh key
2. ssh-copy-id <user>@<remote_ip>
3. Add entry in ~/.ssh/config:
#scriptenginedev
Host lz-scriptenginedev
        HostName 172.20.207.148
        User sas
4. lz-connect.sh - Connect to remote server by group matching
